# Apply to Via Fordham Program Path 1

**Credits:** 3-9

**Start Semester: **Fall, Spring, or Summer

**Campus:** Lincoln Center

Via Fordham is an alternate path to a Fordham University undergraduate degree program for select students who require academic English language support. Path 1 is the best choice for applicants who are interested in attending college in the USA but have not completed the Common App for college admissions.

As a Via Fordham student, you will:

- Strengthen the skills you need to perform well in college-level classes.
- Earn college credits you can put toward your degree.
- Live in New York City, meet locals, and discover American culture.

Spaces in Via Fordham are limited. If you have any questions, please contact us at [[email protected]](/cdn-cgi/l/email-protection#a5d3ccc4e5c3cad7c1cdc4c88bc0c1d0).
