# Enroll in Via Fordham Program Path 2

**Credits:** 3-9

**Start Semester: **Fall, Spring, or Summer

**Campus:** Lincoln Center

*Via Fordham* is an alternate path to a Fordham University undergraduate degree program for select students who require academic English language support. Path 2 is the best choice for students who have already completed the Common App and applied to Fordham.

As a Via Fordham student, you will:

- Strengthen the skills you need to perform well in college-level classes.
- Earn college credits you can put toward your degree.
- Live in New York City, meet locals, and discover American culture.

Spaces in Via Fordham are limited. If you have any questions, please contact us at [[email protected]](/cdn-cgi/l/email-protection#96e0fff7d6f0f9e4f2fef7fbb8f3f2e3).
