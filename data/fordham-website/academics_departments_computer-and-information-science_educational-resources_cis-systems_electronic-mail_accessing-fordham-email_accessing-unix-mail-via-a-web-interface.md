# Accessing Unix mail via a Web Interface

You can use your web browser to access your Unix mail. We use SquirrelMail which provides a user-friendly web interface. It is compatible with `alpine`

folders. To access your mail with SquirrelMail, start a web browser on any internet-connected computer.

Users with accounts on storm visit the URL [https://storm.cis.fordham.edu/webmail](https://storm.cis.fordham.edu/webmail).

Users with accounts on erdos visit the URL [https://dsm.dsm.fordham.edu/webmail](https://dsm.dsm.fordham.edu/webmail).

Enter your Unix username and password in the login box to access your mail.
