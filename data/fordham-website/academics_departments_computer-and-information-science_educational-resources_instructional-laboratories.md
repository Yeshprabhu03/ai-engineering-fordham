# CIS Instructional Laboratories

The Computer and Information Sciences department maintains the following computer labs. These labs are used exclusively for CIS students and faculty. In addition to the labs listed below, all CIS students are given a computer account, email address, and web page on storm (RH) or erdos (LC), the main CIS computing servers.

**RH - JMH 302**

Open and staffed Monday - Friday 9 a.m. - 5 p.m.

Contains 31 Dell all-in-ones with 500GB running Linux, Fedora 31. Available software includes Apache Spark, SSH, Filezilla, Python, Office Suite (LibreOffice), Anaconda 3 with pandas, numpy, scipy, matplotlib and seaborn libraries, Jupyter Notebook, Eclipse, Emacs, GIMP, WEKA, MatLab 2019a and Alice 2.4.3 and 3.1.

**RH - JMH 342**

Open and staffed Monday - Friday 9 a.m. - 5 p.m.

Contains 31 Dell all-in-ones with 500 GB, running Windows 10. Available software includes Filezilla, MobaXterm, Office Suite (Microsoft Office 2016), GameMaker Pro 8.1, Alice 2.4.3, Android Studio, VirtualBox, Magnet Axiom, EnCase Forensic Imager, FTK Imager, Rufus, VLC Player, and MD5 and SHA Checksum Utility.

**LC - LL 612**

Open and staffed Monday - Friday 10 a.m. - 6 p.m.

Contains 31 Dell all-in-one PCs, with dual-booting between Linux Fedora 31and Windows 10). Available Linux software includes Matlab 2019a, Gimp, PyCharm, MongoDB and MySQL. Available Windows software includes Alice 2.4/Alice 3.1, GameMaker Studio 1.4 and GameMaker Pro 8.1, Visual Studio, and Android Studio. Windows includes VirtualBox with Kali Linux and Metasploit. The erdos server also has MySQL, MongoDB, along with the following Python libraries in Anaconda 3: pandas, numpy, scipy, matplotlib and seaborn.

**LC - LL 601**

Contains 31 Dell all-in-one PCs, with Windows 10. Available software includes Filezilla, MobaXterm, Office Suite (Microsoft Office 2016), EnCase Forensic Imager, FTK Imager, Rufus, MD5 and SHA Checksum Utility, and Alice 3.1

For any issues including password resets or forgotten login names please [submit a ticket](https://fordhamcis.on.spiceworks.com/portal/tickets).
