# Web Developer Intern

**For Students**

**DIVISION** Strategic Application Solutions

**POSITION** Web Developer Intern

**SUPERVISOR** Charles-Henri Sanson, Sr. Director of Strategic Application Solutions

**RESPONSIBILITIES**

- Participate in programming for interactive websites
- Implement proper data architecture for application websites
- Work with server architecture for application websites
- Work with other SAS staff to execute design requirements
- Follow visual designs established by the Marketing and Communications Department
- Must be fluent in designing customer end user experience interfaces

**QUALIFICATIONS**

- Must be registered Fordham student in good academic standing
- Convert Photoshop design files into standards-compliant HTML code and templates
- Participate in the development of responsive web pages
- Provide application-specific programming assistance
- Help troubleshoot user-related web client issues
- Help troubleshoot application-related issues
- Work within an Enterprise Project Management framework
- Provide ongoing status reports to the SAS Team Leads

**TECHNICAL KNOWLEDGE**

- Complete knowledge of JavaScript, XHTML, CSS, ASPX, Perl/CGI, PHP
- Experience with .NET, JSP, XML, RSS, JSON, ODBC, JDBC, LDAP authentication and CAS authentication
- Extensive knowledge of Dreamweaver, Photoshop, SQL, Word, and Excel
- Experience with Groovy and Grails
- Knowledge of Microsoft Sharepoint is a plus
- Strong verbal, written and interpersonal communication
- Proven analytical, diagnostic, organizational, and problem-solving skills

**COMPENSATION:** $18.00 per hour
