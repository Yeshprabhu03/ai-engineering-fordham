# Papercut Client (Fordham Device)

### Overview

All Fordham managed devices have the Paper Cut client installed. You may need to log into the client the first time to set up the FindMe print queue. Once you are logged in, you can print to the FindMe printer from any application as if it were a local printer.

### Log into the PaperCut Print Deploy Client

- Click on the Papercut icon in the systray (lower right corner for Windows, Upper right corner for Mac)
- Click View my printers
- If prompted to log in, enter your Fordham username and Password
- The client will show you a printer called FindME[Printer](Mobility) with the status installed

### Print to the Papercut Client

- Print a document from any application
- Select the printer destination FindME[Printer](Mobility)
- If you are in a lab environment, select FindME-LAB [Printer](Mobility)
- Note: you may need to select ‘see more’

- Enter the appropriate settings and click print
- If you in a lab environment, you will be prompted to log in for that print job.
- Log in with your Fordham username and Password
