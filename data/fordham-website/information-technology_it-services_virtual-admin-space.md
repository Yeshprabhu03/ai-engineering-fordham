# Virtual Admin Space

**For Faculty, Staff**

## Overview

Fordham's Virtual Admin Space, hosted by Microsoft Azure Virtual Desktop, provides faculty and staff remote access to University resources and applications typically accessed from an office-based computer.

### How to connect

The Virtual Admin Space can be accessed via the *Microsoft Windows App* on your browser or downloaded desktop clients.

Log in with your Fordham credentials on**Browser Client:**[MyDesktop.Fordham.edu](https://mydesktop.fordham.edu/)from any browser using any device.[Browser access instructions.](https://scribehow.com/shared/Fordham_University_Virtual_Academic_and_Virtual_Admin_spaces_Windows_App_Web_and_Client_configuration__336Fn2QST9OCB2-c44i7yg#75617449)**PC Desktop Client**: Download the[Windows App for PC](https://apps.microsoft.com/detail/9n1f85v9t8bn?hl=en-US&gl=US)and follow the[PC installation instructions](https://scribehow.com/shared/Fordham_University_Virtual_Academic_and_Virtual_Admin_spaces_Windows_App_Web_and_Client_configuration__336Fn2QST9OCB2-c44i7yg#b2949b73)to complete setup.**Apple Mac Desktop Client:**Download the[Windows App for Mac](https://apps.apple.com/us/app/windows-app/id1295203466?mt=12)and follow the[Mac OS installation instructions](https://scribehow.com/shared/Fordham_University_Virtual_Academic_and_Virtual_Admin_spaces_Windows_App_Web_and_Client_configuration__336Fn2QST9OCB2-c44i7yg#84cf8111)to complete setup.**Connect to Microsoft OneDrive & Office and Google Drive:**On the Virtual Admin Space, you can optionally[configure cloud storage services such as Microsoft OneDrive and Google Drive](https://scribehow.com/shared/Fordham_University_Virtual_Academic_and_Virtual_Admin_spaces_Windows_App_Web_and_Client_configuration__336Fn2QST9OCB2-c44i7yg#82252603)to save large files and make them readily available from other devices. You can also[sign in to Microsoft Office to activate Office apps](https://scribehow.com/shared/Fordham_University_Virtual_Academic_and_Virtual_Admin_spaces_Windows_App_Web_and_Client_configuration__336Fn2QST9OCB2-c44i7yg#cb04450a)within the space.

Note: Save your work frequently to your desktop or documents folder on your hard drive, especially if your network connection is unreliable. This will ensure access to those files even when your session is disrupted due to a poor network connection.

For an optimal experience, make sure you have a high-speed Internet connection. WiFi may be used, but a wired connection is best. Smartphone hotspots are not recommended.

For help logging in or for technical support, please view the [complete Virtual Space Access Guide](https://scribehow.com/shared/Fordham_University_Virtual_Academic_and_Virtual_Admin_spaces_Windows_App_Web_and_Client_configuration__336Fn2QST9OCB2-c44i7yg#82252603) or contact the [IT Service Desk](https://www.fordham.edu/information-technology/it-services/technology-support/).

Additional software can be added to the virtual office desktop environment by request if the University owns a license to it. Contact the [IT Service Desk](/information-technology/it-services/technology-support/) with your request and Info Tech will review it.
