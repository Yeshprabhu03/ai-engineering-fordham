# Standard Software

**For Students, Faculty, Staff**

## Overview

Fordham University provides free access to licensed software. Depending on the license requirements, software may be available to use on personal devices and/or is available for use on University-issued devices, on campus at the [Student Computer Labs](/information-technology/it-services/academic-technology/computer-teaching-labs/), as well as virtually, via the [Virtual Academic Labs](/information-technology/standard-software/virtual-academic-space/).

## Getting started

[Workshops](https://itnews.blog.fordham.edu/?post_type=tribe_events&tribe_event_display=month)on supported software are offered throughout the semester for faculty and students.- Unless otherwise noted, all web-based applications work best with the latest version of your computer's browser.
- For some software downloads, authentication with a Fordham username and password is required.
