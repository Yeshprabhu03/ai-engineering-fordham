# Tutorials and Presentations

## Financial Aid

##### How to Create an FSA ID

An FSA ID is simply a username and password combination, which can be used to securely log in to U.S. Department of Education websites like [https://StudentAid.gov](https://StudentAid.gov) and[ https://fafsa.gov]( https://fafsa.gov).

##### What is workstudy?

Check out this video from Lauren Albano, M. Ed, of BestColleges.com, to learn about how work-study can help you pay for college. Visit [StudentAid.gov/workstudy](https://StudentAid.gov/workstudy) to learn more.
