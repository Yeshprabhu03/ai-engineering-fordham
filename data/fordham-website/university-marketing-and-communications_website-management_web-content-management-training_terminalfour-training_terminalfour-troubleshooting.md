## Items Not Working

-
We recommend that you use either Firefox, Chrome, or Safari when working in

[Terminalfour](https://t4.fordham.edu/).

Please note that Internet Explorer does not allow you to add hyperlinks. -
- Make sure the URL you are providing is an RSS feed.
- Confirm that you have the correct URL for the RSS feed by checking if the page is filled with code.
