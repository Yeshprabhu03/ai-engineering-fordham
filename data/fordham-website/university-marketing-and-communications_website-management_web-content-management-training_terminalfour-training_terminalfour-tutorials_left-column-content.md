# Left/Side Column Content

[General Side Content](#general) | [Buttons](#Button) | [Contact Information](#Information) | [Infographic Text Only](#Infographic)

Side or left-column content provides important information related to the general content found on that page.

Examples of information found in side content are:

[Contact Information](/graduate-school-of-education/admissions/tuition-and-financial-aid/)[Links](/student-life/department-of-transportation/)- Statistics
- Event details
- Lists

## Selecting a Side Content Type

It is important to decide what purpose your side content will serve. Please refer to our tutorials on specific side content types such as:

### To Add General Side Content

- Navigate to the desired page you want to edit
- Select the "Add Content" button
- Adjust the amount of Content Types shown by clicking the Display Records which is set at 10, and change to All
- Alternatively: Type side content in the filter to populate the various side content types

- Scroll down the list of content types, then select side: general content
- Create a name for the content that will be displayed
- In the bottom right, select the "Save Changes" button

### To Add a Button in the Side Content

- Navigate to the page you want to edit
- Click the green "Add Content" button
- Search for "Side: Button"
- Create a descriptive name for this content type
- Give the button linked text, this is what is viewed when on the webpage

Decide if your button will link to an internal (within Fordham.edu) or external destination:

Internal Links:

- Click Add Section Link or Add Content Link
- Find the content you would like to link to
- Click "Save Changes"

External Links:

- Paste the URL of the webpage you would like to link to
- Click "Save Changes"
- Click "Preview" to view your current draft and ensure your link works
- Click "Save and Approve" to publish to the live site

To Add Contact Information in the Side Content

- Navigate to the page want to edit
- Click the green "Add Content" button
- Search for Side: Contact
- Enter a descriptive name
- Insert contact information into the relevant fields

### To Add Infographic Text Only

- Navigate to the page you want to edit
- Click the green "Add Content" button
- Search for Side: Infographic Text Only
- Enter a descriptive name
- Select the background color of the first infographic
- Insert the first statistic
- Give the statistic a title
- Enter a description of the description if necessary

Note: There is a 10 character limit for statistics
