# Web Outage Procedures

When experiencing issues with the website, please follow the below instructions.

- If you notice an issue with
[fordham.edu](https://www.fordham.edu)or[news.fordham.edu](http://news.fordham.edu/), please notify all of the following:[[email protected]](/cdn-cgi/l/email-protection#e2958780918b96878487878680838189a2848d90868a838fcc878697),[[email protected]](/cdn-cgi/l/email-protection#8be6fde2e5e8eee5ffbabbcbede4f9efe3eae6a5eeeffe),[[email protected]](/cdn-cgi/l/email-protection#e3978f8a8d878691a3858c91878b828ecd868796). - Once the above are notified they will immediately contact the CMS vendor as well as open a ticket with Fordham IT Service Desk.
- If the issue is widespread and ongoing, key University stakeholders will be notified such as deans, VPs, and web administrators, and a message will be added to the University website.
