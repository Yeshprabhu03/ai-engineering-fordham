# SSG How to Apply

Applications are accepted via the [SSG Online Application Form](https://docs.google.com/forms/d/e/1FAIpQLSeMPAU5zK1ESZlQScQHsJX-c0GMcyXAsTgzR7wMhzh01b6deA/viewform) and [GSAS and GSC Budget Proposal Form](/media/home/schools/graduate-school-of-arts-sciences/pdfs/GSAS-and-GSC-Budget-Proposal-Form.pdf).

Please refer to the table below for guidance on required materials for each SSG funding track.

Students should include a Fordham faculty reference on the SSG application that the Committee can contact if there are any clarifying questions regarding the proposal. Please email [[email protected]](/cdn-cgi/l/email-protection#4c3f3f2b0c2a233e28242d2162292839) with any questions.
