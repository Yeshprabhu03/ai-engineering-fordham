# Endpoint Protection

Endpoint protection is essential for securing your computer against cyber threats. Fordham deploys [CrowdStrike Falcon](https://www.crowdstrike.com/platform/endpoint-security/) on all University-owned and managed devices, offering advanced security features such as antivirus and malware protection.

For personal devices, we recommend the following free endpoint protection applications:

-
[Microsoft Defender](https://support.microsoft.com/en-us/windows/stay-protected-with-windows-security-2ae0363d-0ada-c064-8b56-6a39afb6a963), which already comes with Windows -
Mac OS X:

[Bitdefender Virus Scanner](https://www.bitdefender.com/solutions/virus-scanner-for-mac.html),[AVG](https://www.avg.com/),[Avast](https://www.avast.com/en-us/free-mac-security)
