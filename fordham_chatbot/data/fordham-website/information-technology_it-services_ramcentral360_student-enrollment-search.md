# Student Enrollment Search

**For Advisors**

## Overview

The Student Enrollment Search feature that allows advisors to conduct in-depth searches of the Hub databases. Advisors can search and filter with multiple combinations of parameters across RamCentral360 to easily look for information.

## Use Student Enrollment Search

![App launcher menu](/media/home/departments-centers-and-offices/fordham-it/Hub-1.png)

- Access RamCentral360 via the Advisor page or My Apps in the portal:
- fordham.edu > My Pages > Advisor > RamCentral360
- fordham.edu > My Apps > Common Tools > RamCentral360

- Click the app launcher (9 dots in top left of menu bar)
- Select
*Student Success Hub* - Select
*Student Enrollment Search*from the drop-down menu

![Home menu dropdown showing Student Enrollment Search selected](/media/home/departments-centers-and-offices/fordham-it/Hub-2-.png)
