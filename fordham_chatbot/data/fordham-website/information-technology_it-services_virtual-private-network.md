# Virtual Private Network

**For Faculty, Staff**

## Overview

Fordham's Virtual Private Network (VPN) provides secure, off-campus access to many of the University's password-protected resources. To access the VPN, you'll need to install the VPN client (FortiClient) on your device.

## Get Connected

To install and set up the VPN software from Ivanti on your computer, click on the relevant link below.

[FortiClient VPN for Windows Installation Instructions](https://docs.google.com/document/d/1X0uY3VJyj-I6uzuzmq9ey8ACrE6Bp8If7JAjVltHM2A/edit?usp=drive_link)[FortiClient VPN for Mac Installation Instructions](https://docs.google.com/document/d/1LxwCJ3qH_8aA8uoEKz7FapdtIs8VnVmEJ5u40_RxW4U/edit?usp=drive_link)

For assistance in downloading or installing the VPN software, or to request additional access, please contact the IT Service Desk at 718-817-3999.
