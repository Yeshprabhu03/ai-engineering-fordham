# Insert Slideshows

- When editing a page and viewing the "Content" tab, click "Add content" in the upper right
- Filter the list for "Gallery" and select the "Gallery" content type
- Complete the following mandatory fields:
- Name
- Select Image
- Image Title
- Image Description

- Click on "Save changes" at the bottom of the page
- Repeat the steps for adding a Gallery content type for each slide you want to have
